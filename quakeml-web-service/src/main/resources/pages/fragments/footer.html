Example for extraction of event data from a geographic area:

https://nnsn.geo.uib.no/nnsn_eqcat/fdsnws/event/1/query?starttime=2020-01-20T00:00:00&endtime=2020-01-25T00:00:00&minlatitude=60.0&maxlatitude=62.0&minlongitude=0.0&maxlongitude=10.0&&format=quakeml20&orderby=time&limit=1000

Note:

  • The query includes a maximum number of events, the search is stopped when this number of events is reached.
  • The response time varies between the formats, text is fastest and quakeml slowest.
  • The date and time selection varies in use on different browsers

The catalog is described in detail in: Jeddi, Z., Sørensen, M.B., Voss, P., Larsen, T.B. and Dahl-Jensen, T. (2018). Seismological monitoring in the Arctic, appendix 4 of Kohnert, K. et al., Catalogue of products and services based on land and terrestrial cryosphere data, INTAROS project, Deliverable 2.9, 64 pp (available under https://intaros.nersc.no/sites/intaros.nersc.no/files/Deliverable_D2.9_final_sgn.pdf).
